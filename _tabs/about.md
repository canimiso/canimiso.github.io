### name: canimiso (canibrain)

![Static Badge](https://img.shields.io/badge/cani-miso-orange)

ゲーム好きの文系だけどtech系が気になる系男子。Debian/FreeBSDの入ったMini-ITX/玄箱HGで自宅サーバー経験あり。

メインPCはIntel Core 2 Duo E6600 (2.40GHz)→Intel Core i5-3570K (3.4GHz, Ivy Bridge)→AMD Ryzen 7 5700X (3.4GHz-4.6GHz)と順当にありがちで無難なCPUをチョイスしてきました。ゲーム好きですがGPUは買い時を逃してGeForce GTX 1070で止まっています。

2Dイラスト/ドット絵も少々嗜みます。

小説は氷と炎の歌(ジョージ・R・R・マーティン) / ウィッチャー(アンドレイ・サプコフスキ) / 銀河英雄伝説(田中芳樹) / 十二国記(小野不由美) / 黄金を抱いて翔べ, 神の火, わが手に拳銃を, リヴィエラを撃て, 地を這う虫(髙村薫)あたりを読み込んでいます。マリア様がみてる(今野緒雪)も全巻持ってるぜ！

ゲームはFallout 3, NV, 4, Oblivion, Skyrim(Bethesda) / Terraria / Path of Exile 1, 2 / Diablo 2, 3 / Torchlight 1, 2 / Grim Dawn / Core Keeper / Elin / Wizardryなどに熱狂。穴掘りゲームとハクスラ、3Dダンジョンに目がないです。
